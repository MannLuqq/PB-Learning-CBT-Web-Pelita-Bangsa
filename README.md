# 📖 Tentang Project

PB-Learning merupakan Sistem Informasi Akademik berbasis web yang dikembangkan untuk **SMP Pelita Bangsa Pamulang**.

Sistem ini membantu proses digitalisasi administrasi sekolah mulai dari pengelolaan data siswa, guru, kelas, absensi hingga Computer Based Test (CBT) dalam satu platform yang terintegrasi.

# ⚙️ Instalasi

Clone repository

```bash
git clone https://github.com/MannLuqq/PB-Learning-CBT-Web-Pelita-Bangsa.git
```

Masuk ke folder project

```bash
cd PB-Learning-CBT-Web-Pelita-Bangsa
```

Install dependency

```bash
composer install
```

Konfigurasi database pada file konfigurasi sesuai lingkungan yang digunakan.

Jalankan server lokal menggunakan Apache/XAMPP atau web server lainnya.
